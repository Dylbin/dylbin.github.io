Sileo for iOS 12.0.x - 15.x (checkra1n & unc0ver).

Changes in Sileo 2.2.5 (09/10/2021)

  • New Page load time improvements and will now show a loading indicator after a reload .
  • Fixed a really dumb memory issue which caused jetsam issues for some people .
  • Fixed the race condition crash from the container queue.
  • Fixed bug where all downloads would start at once potentially causing freezes.

    • Changes in Sileo 2.2.4-1 (08/31/2021)

  • Added support for Header control key removing requirement for it to be in depiction.
  • Added more verbose errors messages for helping debug.
  • Fixed some errors when refreshing.
  • Fixed some problems with black screen when using uicache.
  • Fixed uicache again, hopefully?
  • Fixed some issues with incorrectly formatted sources files.
  • Fixed some items not showing in Settings.
  • Slight backend speed improvements.

    • Changes in Sileo 2.2.3 (08/20/2021)

  • Removed option to reinstall packages that have no repo (local installs).

  • Fixed apt cannot find errors.
  • Fixed UICache behaviour.
  • Fixed crash on launch (Mac).
  • Fixed incorrect version numbers in Installed Packages.


    Changes in Sileo 2.2.1 (08/20/2021)

  • Fixed dpkg lock issues.
  • Fixed archive errors and "More by this developer".
  • Fixed local installation of deb files.

    • Changes in Sileo 2.2 (08/19/2021)

  • Added support for all jailbreaks including checkra1n and unc0ver.
  • Optimizations and performance improvements.
  • Bug Fixes and QOL Improvements.
  • Redesigned package install page designed to show you exactly whats happening.
  • New button which cancels downloads.
  • iCloud profile pictures on featured page is back.
  • New button in package depictions to go to that repo.
  • Added an alert for adding dist repos with special handles for BigBoss and Procursus.
  • Adding "https://bigboss.org/" will add BigBoss correctly for you.
  • Support for Canister on the featured page.
  • Improved trackpad support and support for CMD R and CMD + on the Sources Page.
  • Added the ability to change language in Sileo.
  • Sileo will now fade and animate out when you press respring.
  • Added the ability to add installed packages to the wishlist.
  • Added support for sorting by size in Installed Packages.
  • Holding on a file location in Package Contents gives you the option to copy path and Open in Filza (If it's installed).
  • Added Footer in Sources page which will count added sources.
  • Added option to always show install log.
  • Added a check to see if a repo is online and working, this only happends if you add one repo at a time.
  • Improved error messages and handling.
  • News will only show if there has been an article recently.
  • Will now show a warning if you try to remove a package marked as essential.
  • When pasting in a source it will automatically remove a double https://
  • 3 new app icons, thanks to Doregon and Emma#1024.
